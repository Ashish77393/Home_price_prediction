Home Price Prediction - Project Readme

Project Overview

This Django web application allows users to predict the estimated price of a house based on property details. Users fill a form with features like area, bedrooms, bathrooms, and amenities, and the system calculates a predicted price using a machine learning model.

Technologies Used

  • Backend: Django (Python)
  • Frontend: HTML, CSS (Responsive Design)
  • Machine Learning: Pre-trained model (Pickle file)
  • Deployment: Local Django server / can be deployed to cloud

Features

  • Responsive property input form
  • CSRF-protected POST submission
  • Predicted price displayed in a clean, centered card
  • Navigation header with links (Home, About, Readme)

Installation

  1. Clone the repository: git clone https://github.com/username/home-price-prediction.git
  2. Create and activate a virtual environment: python -m venv env
    Windows: env\Scripts\activate
    Linux/Mac: source env/bin/activate
  3. Install dependencies: pip install -r requirements.txt
  4. Run migrations: python manage.py migrate
  5. Start the server: python manage.py runserver
  6. Open http://127.0.0.1:8000/ in your browser

Usage

  • Navigate to Home page
  • Fill in the Property Details Form
  • Click Submit
  • View the predicted home price on the result page
